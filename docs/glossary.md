# Glossary

**Adverse impact** — Unintended harm to a specific user group.  
**Explainability** — Clear reason(s) for a given result or rank.  
**Human-in-the-loop** — A person can review or override AI output.  
**Model card** — A short summary of a model’s purpose, risks, and limits.  
**Redaction** — Removing or masking sensitive details before publishing.
