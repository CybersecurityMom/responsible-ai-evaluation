# Examples (How to Use This Toolkit)

- **Persona relevance test:** Create two personas with different goals and verify top-5 results match their intent.
- **Location override test:** Set a city + radius and confirm unrelated locations don’t appear.
- **Deduplication test:** Confirm duplicate items don’t repeat across pages.
- **Disclosure clarity test:** Ask 3 users to rate the pre-evaluation disclosure (1–5) and record feedback.
- **Explainability test:** Verify the presence of short “reason strings” for each result.

Add redacted screenshots or notes under `/evidence/<system>/`.
